    关于 VirtualDOM 和 Diff

    • 在现代 web 应用程序中使用 JavaScript 操作 DOM 是必不可少且频繁的
    • 但是这个操作是非常消耗性能的，因为 JavaScript 操作 DOM 对象要比操作其他对象要慢得多
    • 大多数 JavaScript 框架对于 DOM 的更新远远超过其必须进行的更新，从而使得这种缓慢操作变得更糟
